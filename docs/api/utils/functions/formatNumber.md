# Function: formatNumber()

> **formatNumber**(`n`): `string`

Formats a number using US locale formatting

## Parameters

| Parameter | Type | Description |
| ------ | ------ | ------ |
| `n` | `number` | Number to format |

## Returns

`string`

Formatted number string

## Example

```ts
formatNumber(1234567.89); // "1,234,568"
```
